🪙 Cara ou Coroa

Projeto acadêmico de um jogo Cara ou Coroa desenvolvido para web utilizando Python, HTML, CSS e JavaScript.
O sistema permite cadastrar jogadores, realizar partidas e registrar os resultados em um banco de dados.

O objetivo do projeto é praticar conceitos básicos de desenvolvimento web full-stack, comunicação entre frontend e backend, e manipulação de dados em SQLite, sem utilizar frameworks.

🚀 Funcionalidades

Cadastro de jogadores

Seleção de dois jogadores para a partida

Simulação de moeda (cara ou coroa)

Resultado aleatório da jogada

Registro das partidas no banco de dados

Página de histórico de jogos

Página de estatísticas

📊 Estatísticas registradas

O sistema também registra algumas informações sobre os jogos:

Quantidade de caras e coroas

Total de partidas realizadas

Jogadores que mais participaram

Curso e turno com mais jogos

🛠 Tecnologias utilizadas

HTML5
